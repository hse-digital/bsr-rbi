Class 2 building inspector Assessing plans: which category of building plans are you competent to assess?

Competent means you can assess building plans, drawings and associated information (for example, material specifications, structural calculations and fire strategies) to ensure they comply with building regulations.

To register as a class 4 technical manager you must be competent to do all BSR regulated activities (for example, assessing plans and inspections) for all categories of class 2 or class 3 building inspectors.
