Competency assessment certificate

To do BSR-regulated activities unsupervised, you must have passed a BSR-approved competency assessment for the class you want to register for.

Assessment is done by BSR-approved providers:

If you have not passed assessment, you can still complete your registration, but you will be registered as a class 1 building inspector. You will not be able to do BSR-regulated building control activities without supervision after April 2024.

You can request to change your class of registration when you have completed a relevant competency assessment.
